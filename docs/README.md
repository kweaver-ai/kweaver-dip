# 文档导航

本页是 KWeaver DIP 仓库文档入口，帮助开源社区快速定位所需信息。

## 新用户入口

- 项目总览：`../README.zh.md`
- 英文总览：`../README.md`
- 目录结构：`./PROJECT_STRUCTURE.md`
- 文档规划：`./DOCUMENTATION_PLAN.md`

## 按角色阅读

- **产品与业务同学**
  - 先读 `../README.zh.md` 的项目介绍与目标
  - 再读 `./PROJECT_STRUCTURE.md` 了解模块职责
- **开发者**
  - 先读 `./PROJECT_STRUCTURE.md`
  - 再进入具体模块目录查看模块级 `README.md`
- **开源贡献者**
  - 先读仓库根 `README`
  - 再查看 `rules/` 与 `release-notes/`

## 文档维护约定

- 新增模块必须补充模块级 `README.md`。
- 目录职责变更必须同步更新 `./PROJECT_STRUCTURE.md`。
- 每个可见版本都应在 `release-notes/` 留下变更记录。
