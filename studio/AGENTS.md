开发数字员工运营平台 DIP Studio 的后端服务

## 技术要求
- 使用 Python 3.14 作为开发语言
- 严格遵循 PEP 8 规范编写代码
- 使用 FastAPI 作为 HTTP 框架
- 使用 uv 进行包管理
